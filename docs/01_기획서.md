## 3. 타겟 사용자

**성동구(성수·왕십리·행당 등)에서 카공하는 대학생 및 직장인**

- 노트북 배터리가 닳을까 봐 콘센트 자리 확보에 스트레스를 받는 사람

## 4. 핵심 문제 & 해결책

| 문제 (사용자가 겪는 불편) | 해결책 (우리가 제공하는 것) |
|---|---|
| 카공 가능한 카페인지 직접 가봐야 알 수 있다 | 네이버 리뷰 AI 분석으로 사전에 5대 지표 점수 제공 |
| 콘센트·와이파이·소음 정보가 리뷰에 흩어져 있다 | Qwen3.5 LLM이 리뷰 텍스트를 정제·점수화해 한눈에 표시 |
| "카공하기 좋다"는 기준이 사람마다 다르다 | 콘센트·와이파이·소음·눈치·테이블 5개 지표로 세분화해 직접 필터링 가능 |

## 5. 핵심 기능

### ✅ 필수 (MVP)
- **카페 목록 조회** — 성동구 카페를 5대 지표 기준으로 정렬·필터링
- **카페 지도 표시** — 카카오맵에 점수별 마커 표시
- **카페 상세 조회** — 카페별 5대 지표 점수 및 수집 리뷰 수 표시
- **지표 필터링** — 콘센트 4점↑, 와이파이 4점↑ 등 조건 설정

### 💡 권장 (시간 남으면)
- **검색** — 카페 이름으로 검색
- **지표별 랭킹** — 콘센트 TOP 10, 와이파이 TOP 10 등

## 6. 사용자 시나리오

```
1. 사용자가 메인 페이지에 접속한다.
2. 성동구 카페가 카카오맵 위에 점수별 마커로 표시된다.
3. 우측 필터 패널에서 "콘센트 4점↑", "조용한 카페" 조건을 설정한다.
4. 조건에 맞는 카페만 지도와 목록에 필터링되어 나타난다.
5. 마음에 드는 카페 마커를 클릭하면 5대 지표 점수 카드가 팝업으로 뜬다.
6. 카페를 선택해 상세 페이지로 이동하면 총 리뷰 수와 점수 상세를 확인한다.
```

## 7. 화면 구성

| 페이지 | 경로 | 핵심 요소 |
|---|---|---|
| 메인 (지도) | `/` | 카카오맵 + 카페 마커 + 필터 패널 |
| 카페 목록 | `/list` | 점수 기준 카드 리스트 + 정렬·필터 |
| 카페 상세 | `/cafe/:id` | 5대 지표 점수 차트 + 기본 정보 |
